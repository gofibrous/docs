## Getting Started

1. **Choose your network** - Select from Base, Scroll, Starknet, or HyperEVM
2. **Check API health** - Use the health check endpoint to verify connectivity
3. **Find optimal routes** - Query the route endpoint for best trading paths
4. **Execute transactions** - Use execute or calldata endpoints to complete swaps

## Rate Limits

All API endpoints are rate-limited to ensure fair usage. Please refer to our [rate limiting documentation](/api-reference/rate-limit) for specific details.

## Authentication

Currently, no authentication is required for public API endpoints. All requests are processed without API keys.
