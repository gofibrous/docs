# Install Fibrous Router Sdk

<CodeGroup>
```bash npm
npm install fibrous-router-sdk
```

```bash yarn
yarn add fibrous-router-sdk
```

```bash pnpm
pnpm add fibrous-router-sdk
```
</CodeGroup>

The Fibrous Router SDK provides a simple and efficient way to interact with the Fibrous Smart Order Router API. It includes functionality for:

- Getting supported tokens and protocols
- Finding the best routes for token swaps
- Executing single and batch swaps
- Building and sending transactions

## Quick Start

Here's a simple example of how to initialize the SDK:

```typescript
import { Router as FibrousRouter } from "fibrous-router-sdk";

const router = new FibrousRouter();
```

## Features

- **Token Support**: Easy access to supported tokens across different chains
- **Protocol Integration**: Seamless integration with multiple DEX protocols
- **Route Optimization**: Advanced algorithms to find the best swap routes
- **Transaction Building**: Helper functions to build and execute transactions
- **Type Safety**: Full TypeScript support with comprehensive type definitions
