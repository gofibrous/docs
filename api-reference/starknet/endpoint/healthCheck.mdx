https://api.fibrous.finance/starknet

Verify if the Fibrous Finance Starknet API is up and running. This endpoint can be used for monitoring and ensuring the API service is healthy.

## Response

<ResponseField name="status" type="string">
  The health status of the API. Will be "ok" if the service is healthy.
</ResponseField>

<ResponseField name="version" type="string">
  The current version of the API.
</ResponseField>
