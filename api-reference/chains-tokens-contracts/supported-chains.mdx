# Supported Chains

Fibrous Finance provides optimal token swap routing across multiple blockchain networks. Each network is fully integrated with our routing algorithm to ensure the best possible trading experience.

## Mainnets

| Chain Name | Chain ID | Chain Tag |
|------------|----------|-----------|
| **Starknet** | SN_MAIN | `starknet` |
| **Base** | 8453 | `base` |
| **Scroll** | 534352 | `scroll` |
| **HyperEVM** | 998 | `hyperevm` |
| **Monad** | 41454 | `monad` |

## Integration Support

### API Support
All chains are fully supported through our unified API:
- ✅ Route optimization
- ✅ Swap execution
- ✅ Calldata generation
- ✅ Health monitoring

### SDK Support
Our TypeScript SDK supports all chains:
```typescript
import { Router as FibrousRouter } from "fibrous-router-sdk";

const fibrous = new FibrousRouter();
const chains = fibrous.supportedChains;

// Supported chains:
// - starknet (SN_MAIN)
// - base (8453)
// - scroll (534352)
// - hyperevm (998)
// - monad (41454)
```

### Smart Contract Support
Router contracts are deployed on all supported chains. See [Contract Addresses](/api-reference/chains-tokens-contracts/contract-addresses) for details.
