FibX supports four EVM-compatible chains. The default chain for all commands is **Base**.

## Chain Overview

| Chain    | Native Token | Chain ID | CLI Flag           | Aave V3 |
| -------- | ------------ | -------- | ------------------ | ------- |
| Base     | ETH          | 8453     | `--chain base`     | Yes     |
| Citrea   | cBTC         | 4114     | `--chain citrea`   | No      |
| HyperEVM | HYPE         | 999      | `--chain hyperevm` | No      |
| Monad    | MON          | 10143    | `--chain monad`    | No      |

## Chain Details

  <Accordion title="Base" icon="ethereum">

| Tx Status        | Yes  | Yes    | Yes      | Yes   |
| JSON Output      | Yes  | Yes    | Yes      | Yes   |

## Selecting a Chain

Pass the `--chain` flag to any command. If omitted, Base is used.

```bash
# Explicit chain selection
npx fibx@latest trade 0.1 ETH USDC --chain base
npx fibx@latest balance --chain monad

# Shorthand with -c
npx fibx@latest balance -c citrea
```
